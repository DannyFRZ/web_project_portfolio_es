# web_project_portfolio_es

## Descripción del Proyecto

Este proyecto es un portafolio web interactivo diseñado no solo para mostrar mi trabajo, sino también para servir como una herramienta de prueba visual. Su funcionalidad principal permite alternar y evaluar cómo se comporta la interfaz en **diferentes dimensiones de dispositivos**, enfocándose especialmente en pantallas de teléfonos móviles y tabletas.

El objetivo central es garantizar que el **diseño responsivo** se mantenga impecable en cualquier resolución, ofreciendo una experiencia de usuario fluida, adaptabilidad en tiempo real y consistencia visual sin importar el tamaño de la pantalla.

---

## Tecnologías y Técnicas Utilizadas

Para lograr una réplica exacta y fiel del diseño planificado en **Figma**, se implementaron las siguientes tecnologías y metodologías de desarrollo:

- **Media Queries de CSS (`@media`):** La técnica fundamental para el diseño responsivo. Se utilizaron puntos de quiebre (_breakpoints_) estratégicos para aplicar especificaciones técnicas y estilos exclusivos a cada rango de pantalla (móvil, tableta y escritorio). Esto asegura que la disposición de los elementos se reajuste de manera limpia y ordenada.
- **Fidelidad de Diseño (Figma to Code):** Todo el desarrollo visual se guio minuciosamente por un prototipo de Figma, traduciendo espaciados, tipografías y componentes interactivos para lograr una visualización pulida y profesional.
- **Diseño Fluido y Flexible:** Más allá de los puntos de quiebre fijos, se emplearon unidades relativas y propiedades de caja flexible (como Flexbox o CSS Grid) para que la transición entre dimensiones sea orgánica y no rompa la estructura en ningún dispositivo intermedio.
